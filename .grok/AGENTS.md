## Skills — à la demande uniquement

- **Site web générique** → aucune skill, coder normalement.
- **Site holding / Privy / « comme Kikou »** → lire uniquement `.grok/skills/spawn-holding-site/SKILL.md` (ou `/spawn-holding-site`).
- **Ne pas** charger plusieurs skills ni `site-fonctionnel.md` sans demande holding explicite.

Catalogue template : `.grok/skills/` — chaque skill a `disable-model-invocation: true`.  
Skills métier : repo `aria-skills` (statut non vérifié cette session — ne pas dupliquer ici).

## Conventions

- Modifier uniquement le code requis par la tâche.
- Respecter le style, les types et les abstractions existants.
- Réutiliser les fonctions et composants déjà présents plutôt que de réimplémenter.
