# Template Grok + Cursor

Toujours répondre en français.

## À propos de ce dépôt

Point de départ pour tout nouveau projet avec Grok Build et Cursor.
Les règles dans `.cursor/rules/` et `.grok/` sont chargées automatiquement.

## Premier lancement sur un nouveau PC

1. Cloner ou créer un projet depuis ce template GitHub.
2. Copier la règle globale française vers votre profil :
   - `~/.cursor/rules/francais.md` (Windows : `%USERPROFILE%\.cursor\rules\francais.md`)
   - `~/.grok/AGENTS.md` avec le contenu de `.grok/AGENTS.md`
3. Optionnel : configurer `~/.grok/config.toml` à partir de `.grok/config.toml.example`
